HICHEEL_21_4.06 - TURBOREPO + NEXT.JS MONOREPO

WHAT IS INCLUDED
- apps/web   -> main web app
- apps/admin -> admin panel
- packages/ui -> shared Button component

RUN
1) npm install -g pnpm
2) cd hicheel_21_4.06_monorepo_next
3) pnpm install

START WEB
pnpm dev:web
Browser: http://localhost:3000

START ADMIN
Open new terminal:
pnpm dev:admin
Browser: http://localhost:3001

OR START BOTH
pnpm dev

IMPORTANT
If teacher asks for pnpm create turbo@latest, say this ZIP is the same monorepo structure prepared manually for fast demo.
